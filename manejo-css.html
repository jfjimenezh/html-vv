SINTAXIS

un conjunto de reglas consta de un selector y un bloque de de declaracion.El selector apunta al elemento HTML que desea diseñar. El bloque de declaración contiene una o más declaraciones separadas por punto y coma. Cada declaración incluye un nombre de propiedad CSS y un valor, separados por dos puntos. Una declaración CSS siempre termina con un punto y coma, y ​​los bloques de declaración están rodeados por llaves.
